# CSharp--in-30-Days

Welcome to **CSharp--in-30-Days** 👋  
This is a structured, day-by-day learning project where I explore core and advanced concepts of the C# programming language.

Each day covers a focused topic, complete with hands-on practice and beginner-friendly explanations.  
If you're new to C#, or want to brush up your skills — this repo is for you!
